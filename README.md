## Tamyuz Market

#### Technologies
- Java 25
- Spring Boot 4.0.0
- Spring Data JPA
- Spring Web
- Spring Security
- Spring Boot DevTools
- Flyway Migration
- Spring Boot Actuator

#### Implemented Business Logic
- User Registration
- User Login
- CRUD Operations on Products
- Update Item
- Create Order

#### How to run the project
- install docker desktop
- run: ```cd docker```
- run: ```docker-compose up -d```
